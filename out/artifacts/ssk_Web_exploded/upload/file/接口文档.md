**通用约定**

1. 正式接口：http://localhost:8080
2. 测试接口：http://localhost:8080

------




#### 接口文档

```http
http://localhost:8080/ssk/if/selectInterface
```

#### 返回值

- data 对象数组
- id 编号id
- name 接口名
- url 接口下载地址
- pageUrl 页面展示地址
- gmtCreate 主动创建（13位时间戳）
- gmtModified 被动更新（13位时间戳）
- isDeleted 是否删除
