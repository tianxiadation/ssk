#### 1.时间同步

```http
https://exam.com/system/get_system_time
```

#### post参数

| 变量名 | 类型   | 是否必填 | 注释     |
| ------ | ------ | -------- | -------- |
| uuid   | String | 是       | 设备编号 |
|        |        |          |          |
|        |        |          |          |

#### 返回值

- data 服务器系统的时间（13位时间戳）

#### example

```json
{
    "code": 200,
    "msg": "success",
    "data": 1400000000000
}
```
